## Toshitext
Send bitcoin via SMS the naive way with Twilio, Blockcypher, Flask and Google Sheets

## Please Do Not Use In Production
This is just a proof of concept.

+ Bitcoin 
+ Twilio
+ Blockcypher API
+ Blockcypher Block Explorer
+ Google Cloud
+ Heroku 
+ Google Cloud SQL
+ Google Drive API
+ Google Secrets Management
+ Python Flask
+ Coinbase API
